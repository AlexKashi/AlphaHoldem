# EasyRL

## Installation
```bash
git clone https://github.com/taochenshh/easyrl.git
cd easyrl
conda env create -f environment.yml
conda activate easyrl
```

Checkout examples [here](https://github.com/taochenshh/easyrl/tree/master/examples)
